## Installation

1. Clone the repository.
   
   ```sh
   git clone https://github.com/ottohellwig/imdb-clone.git
   ```

2. Install the dependencies.

   ```sh
   npm install
   ```

3. Deploy local version.

   ```sh
   npm start
   ```
